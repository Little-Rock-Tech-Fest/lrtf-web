Little Rock Tech Fest
==============

This is the current website for Little Rock Tech Fest

http://www.lrtechfest.com/

### Prerequisites

[Node.js](https://nodejs.org/en/)

[MongoDB](https://www.mongodb.com/)

### Install

```
npm install
cp .env.example .env
```

### Launch

```
node keystone
```

http://localhost:3000
