Little Rock Tech Fest
==============

This is the (dev) website for Little Rock Tech Fest

https://lrtf-2019.herokuapp.com/

### Prerequisites

[Node.js](https://nodejs.org/en/)

[MongoDB](https://www.mongodb.com/)

### Install

```
npm install
cp .env.example .env
```

### Launch

```
node keystone
```

http://localhost:3000
